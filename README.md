# 📡 Análise de Cobertura Wi-Fi com NetSpot

## 📌 Descrição
Este projeto apresenta um estudo completo de análise de rede Wi-Fi em ambiente residencial, avaliando qualidade de sinal, interferências e desempenho da rede.

## 🎯 Objetivo
Identificar falhas de cobertura, interferência e desempenho, propondo melhorias para otimização da rede sem fio.

## 🛠️ Ferramentas Utilizadas
- NetSpot
- Teste de Velocidade (Speed Test)
- Adaptador Wi-Fi TP-Link AC1300
- Roteador Nokia G-1425G-A

## 📊 Parâmetros analisados

### 📡 RSSI (Intensidade do Sinal)
- Excelente próximo ao roteador
- Queda significativa em áreas distantes

### 📶 SNR (Relação Sinal-Ruído)
- Valores entre 56 dB e 60 dB
- Considerado excelente para uso estável :contentReference[oaicite:0]{index=0}

### ⚠️ Interferência (SIR)
- Banda 2.4 GHz com alta interferência
- Sobreposição de canais (1, 2, 3 e 4) :contentReference[oaicite:1]{index=1}

### 🚀 Velocidade
- 5 GHz: até 231 Mbps download
- 2.4 GHz: até 42 Mbps download :contentReference[oaicite:2]{index=2}

## 🔍 Metodologia
- Ambiente residencial (~46m²)
- 22 pontos de medição distribuídos
- Análise com planta baixa no NetSpot :contentReference[oaicite:3]{index=3}

## 📈 Resultados
- Excelente desempenho próximo ao roteador
- Perda de sinal em áreas com obstáculos
- Interferência significativa na banda 2.4 GHz

## 🧠 Propostas de melhoria
- Reposicionamento do roteador para área central
- Atualização de hardware
- Melhor escolha de canal Wi-Fi :contentReference[oaicite:4]{index=4}
